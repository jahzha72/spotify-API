# Spotify Explorer Project

A modern, responsive web application that integrates the **Spotify Web API** to allow users to search for tracks, albums, and playlists in real-time.

## üöÄ Features
- **OAuth 2.0 Authentication**: Implements the Client Credentials Flow to securely fetch access tokens.
- **Dynamic Search**: Real-time retrieval of music data including track titles, artist names, and high-quality artwork.
- **Audio Preview**: Integrated HTML5 Audio player to stream 30-second song previews directly from Spotify's servers.
- **Defensive Programming**: Robust error handling to manage null data, missing images, and API timeouts.
- **Responsive UI**: A "Spotify-inspired" dark theme built with CSS Grid and Flexbox, fully optimized for mobile, tablet, and desktop views.
